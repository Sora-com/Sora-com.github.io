Detailed Description

Módulo encargado de la creación del tablero, tanto por parte del jugador como en modo automático (CPU).

Este módulo gestiona la creación del tablero, permitiendo al jugador configurar sus barcos manualmente o dejando que el sistema los coloque de forma automática para la CPU.

◆ inicioCrearTablero()

void inicioCrearTablero ( Config * config,
int empezarPartida )

Inicia el proceso de creación de los tableros de los dos jugadores.

Según la configuración, permite a los jugadores crear sus tableros manualmente o de forma automática. Si se indica, tras crear ambos tableros se inicia directamente la partida.

El tablero utiliza el siguiente código numérico para sus casillas:

  • 0 = Agua
  • 1 = AguaTocada
  • 2 = Barco
  • 3 = BarcoTocado
  • 4 = BarcoHundido
Parameters
configPuntero a la estructura de configuración del juego.
empezarPartidaIndicador (1 para empezar la partida después de crear tableros, 0 para solo configurar).
