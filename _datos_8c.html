int BarcoHundido ( int *** tablero,
int size,
Coordenada posBarco,
Jugador * jugador )

Verifica si un barco ha sido hundido en el tablero de juego.

Esta función determina si un barco ha sido completamente hundido al comprobar sus partes adyacentes y si no hay partes del barco que aún puedan ser tocadas. Si el barco ha sido hundido, se actualiza el tablero y se incrementa el contador de barcos hundidos del jugador.

Parameters
tableroPuntero a un tablero de juego en forma de matriz tridimensional.
sizeTamaño del tablero.
posBarcoCoordenada (x, y) que indica una de las partes del barco que se ha tocado.
jugadorPuntero a la estructura del jugador cuyo contador de barcos hundidos se actualizará.
Returns
Devuelve 1 si el barco ha sido completamente hundido, 0 si no, y -1 si no se puede determinar.

Cierra el programa y libera la memoria dinámica.

Esta función muestra un mensaje final, espera a que el usuario presione una tecla y luego libera toda la memoria dinámica asignada para las estructuras del juego. También limpia la pantalla antes de terminar el programa.

Parameters
configUn puntero a la estructura Config que contiene los datos de configuración del juego.
Precondition
El juego debe haber terminado antes de llamar a esta función.
Postcondition
El programa se cierra correctamente y la memoria dinámica se libera.

◆ ComprobarGanador()

int ComprobarGanador ( int ** Tablero,
int size )

Comprueba si hay un ganador en el juego.

Esta función recorre el tablero para verificar si todos los barcos han sido hundidos. Si todos los barcos han sido tocados o hundidos, la función retorna 1 (indicando que hay un ganador). Si al menos un barco no ha sido tocado, retorna 0.

Parameters
TableroUn puntero a una matriz que representa el estado del tablero de juego.
sizeEl tamaño del tablero (debe ser cuadrado).
Returns
Retorna 1 si todos los barcos están hundidos, 0 en caso contrario.

Cuenta cuántas celdas de un tipo específico están presentes en el tablero.

Esta función recorre todo el tablero y cuenta cuántas veces se encuentra una celda con el valor especificado por el parámetro "tipo". Esto se utiliza para contar diferentes elementos en el tablero, como los barcos o los disparos.

Parameters
tableroUn puntero a la matriz que representa el tablero de juego.
sizeEl tamaño del tablero (debe ser cuadrado).
tipoEl valor que se va a contar en el tablero (por ejemplo, un barco o un disparo).
Returns
El número de celdas que coinciden con el valor de "tipo".

Detecta si hay un tipo específico de casilla adyacente a una posición.

Esta función verifica si existe una casilla adyacente a la posición especificada en el tablero que coincida con el tipo de casilla proporcionado. Las casillas adyacentes incluyen las diagonales y las casillas laterales.

Returns
1 si se encuentra una casilla adyacente del tipo indicado, 0 en caso contrario.

◆ DibujarTablero()

void DibujarTablero ( int ** Tablero,
int size,
int esContrincante )

Dibuja el tablero en la terminal.

Esta función dibuja el tablero de juego en la terminal, representando diferentes estados de las casillas mediante símbolos y colores. Los posibles estados de las casillas son: AGUA, AGUA TOCADA, BARCO, BARCO TOCADO, y BARCO HUNDIDO. El tablero se dibuja con la representación adecuada según si es el tablero del contrincante o no.

Parameters
TableroPuntero a la matriz que representa el tablero de juego.
sizeEl tamaño del tablero (tanto en filas como en columnas).
esContrincanteIndica si el tablero que se está dibujando es del contrincante (1) o del jugador (0).
