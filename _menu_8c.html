int BarcosConfigPosible ( Barco * barcos,
int numBarcos,
int size )

Verifica si es posible colocar todos los barcos en el tablero según las reglas del juego.

Parameters
barcosUn array de estructuras Barco que contiene información sobre los barcos.
numBarcosNúmero de barcos a colocar en el tablero.
sizeTamaño del tablero (en casillas).
Returns
Devuelve 1 si es posible colocar los barcos, 0 si no lo es.
Precondition
barcos debe contener información válida sobre los barcos seleccionados, numBarcos > 0, size > 0.
Postcondition
Devuelve 1 si es posible colocar todos los barcos en el tablero respetando las reglas, 0 si no lo es.

◆ MenuBarcos()

void MenuBarcos ( Config * config)

Muestra el menú para seleccionar los barcos con los que se jugará.

Esta función permite al usuario seleccionar los barcos con los que quiere jugar. Muestra una lista de los barcos disponibles y permite al usuario elegir cuántos barcos desea añadir, así como la opción de agregar más de un barco de un mismo tipo. La función valida que la selección de barcos no exceda el límite y comprueba si la configuración es posible antes de finalizar.

Parameters
configPuntero a la estructura Config que contiene la configuración actual del juego.

Esta función elimina la configuración actual de los jugadores, el tablero y los barcos. Borra los nombres de los jugadores, los tipos de disparo y tablero, y libera la memoria asignada para los barcos.

configPuntero a la estructura Config que contiene la configuración actual del juego.

Esta función muestra la configuración de los jugadores (nombres, tipo de disparo, tipo de tablero) y los barcos seleccionados para jugar, incluyendo la longitud de cada barco. Además, muestra la dimensión del tablero y el jugador que comenzará el juego.
